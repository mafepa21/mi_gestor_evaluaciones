# Matriz inicial de paridad macOS

| Feature | Fuente primaria | Estado base macOS | Clasificación |
| --- | --- | --- | --- |
| Dashboard | `kmp/iosApp` | Shell operativa con bridge | `apple-adapter-needed` |
| Cuaderno | `kmp/iosApp` + KMP | Pendiente de vista nativa completa | `apple-adapter-needed` |
| Planificación | `kmp/iosApp` + KMP | Pendiente de vista nativa completa | `apple-adapter-needed` |
| Alumnado | `kmp/iosApp` + KMP | Pendiente de vista nativa completa | `apple-adapter-needed` |
| Rúbricas | `kmp/iosApp` + KMP | Pendiente de vista nativa completa | `apple-adapter-needed` |
| Sync LAN | `desktopApp` + `KmpBridge` | Shell y observabilidad visibles | `desktop-only-to-preserve` |
| Backups | `desktopApp` | Servicio macOS nativo añadido | `desktop-only-to-preserve` |
| Informes | `kmp/iosApp` + KMP | Pendiente de UI final | `shared-ready` |
| Ajustes/Comandos | Apple shell | Base implementada | `shared-ready` |
